# AlEx AI Platform v2

Enterprise AI Agent Platform.

## Stack

- Java 21
- Spring Boot 3.5
- Spring Boot static frontend
- Ollama
- Docker

## Architecture

Multi-agent AI platform with RAG and memory.

## Status

Under development.

## Como iniciar

1. Copie e ajuste as variáveis em `.env.example` para `.env`.
2. Execute `./start-v2.sh` para construir e rodar localmente.
3. Para rodar em Docker, use `./start-v2.sh docker`.
